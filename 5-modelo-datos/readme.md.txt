# Modelo de Datos (Entidad-Relación)

En esta carpeta se presenta un modelo de datos para una tienda en línea.

Entidades:
- Productos
- Clientes
- Pedidos
- Métodos de pago

Relaciones:
- Un cliente puede tener múltiples pedidos.
- Un pedido tiene muchos productos.
- Un pedido se paga con un método de pago.
