# Proyecto de Gestión de Bases de Datos Relacionales

Este repositorio contiene un proyecto personal que demuestra el dominio de las tecnologías y competencias técnicas adquiridas en el curso de gestión de bases de datos relacionales.

Se implementan los siguientes temas:
- Conceptos básicos de bases de datos relacionales.
- Consultas SQL (DQL).
- Manipulación de datos (DML).
- Lenguaje de definición de datos (DDL).
- Diseño de modelos de datos (Entidad-Relación).

Cada carpeta contiene ejemplos prácticos y documentación detallada.
